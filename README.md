# detection-engineering
Security detection repository focused on identifying common and emerging attack techniques across identity, cloud, and AI environments.

This repository contains KQL-based detections designed for Microsoft Sentinel and related security telemetry sources. The focus is on practical threat detection patterns seen in real-world environments, including identity abuse, privilege escalation attempts, and AI-assisted attack techniques. Each detection is designed to be readable, adaptable, and tuned for real operational use rather than purely theoretical scenarios.
